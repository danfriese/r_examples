Spatial analysis projects written in R, developed as part of coursework 
at Oregon State University.

# Healthcare Deserts in Oregon
Identifies census block groups in Oregon lacking adequate access to acute care 
hospitals using straight-line distance and a population-weighted composite index. 
Includes spatial autocorrelation analysis (Global and Local Moran's I).

# Heat Vulnerability Index – Eugene/Springfield, OR
Builds a heat vulnerability index (HVI) at the census block group level using ACS 
socioeconomic variables, land surface temperature, and tree canopy cover. Identifies 
statistically significant hot spots using Getis-Ord Gi* analysis and assesses walking 
distance access to cooling centers.
